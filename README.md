 ### 依赖文档
```
cerate-react-app  ele-app
cd ele-app
yarn start 运行项目
yarn eject  不可逆操作暴露webpack相关配置文件（config）
还需要安装的依赖
yarn add react-redux redux react-router-dom react less less-loader
```
- Home 首页
  - Header 头部位置查找
  - Banner  轮播图
  - Tab 底部固定的导航栏
  - GoodsList   商品列表
- Find 发现页面
- Myprofile 个人中心页面
- Order 订单登陆页面
- Search 搜索展示页面
